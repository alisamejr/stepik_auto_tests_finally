Pet‑проект: Автоматизация тестирования интернет-магазина
Стек: Python, Selenium, Pytest, PageObject, Git

Спроектировала фреймворк для UI-тестирования с использованием паттерна PageObject (базовый класс + страницы товара, корзины, логина).

Реализовала методы для позитивных и негативных проверок: is_not_element_present, is_disappeared с явными ожиданиями.

Настроила параметризованные тесты для 10 промо-акций (offer0 – offer9), что помогло обнаружить баг на offer7.

Добавила автоматическую регистрацию пользователей через фикстуры с генерацией уникальных email (на основе time.time()).

Интегрировала поддержку мультиязычности (--language) и выбора браузера (--browser_name) через опции командной строки.

Организовала запуск только ревью-тестов с помощью маркеров @pytest.mark.need_review.
